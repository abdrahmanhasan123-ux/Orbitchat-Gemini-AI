#  OrbitChat: Gemini 1.5 AI Assistant

OrbitChat is a modern, web-based chatbot interface built with **Python** and **Streamlit**. It leverages the power of the **Google Gemini 1.5 Pro** model to provide intelligent, real-time responses.

###  Key Features
* **AI Conversations:** High-speed streaming responses from Gemini 1.5.
* **Modern UI:** A clean, responsive chat interface designed with Streamlit.
* **Smart Memory:** Keeps track of chat history during your session.
* **Cloud Infrastructure:** Developed and optimized for Google Colab and tunneling services.

###  Technical Stack
* **Language:** Python 3.10+
* **Frontend:** Streamlit
* **AI Backend:** Google Generative AI (Gemini API)
* **DevOps:** Google Colab, Cloudflare Tunnels (for remote hosting)

###  How to Get Started
1. **Clone the repo:** `git clone https://github.com/YOUR-USERNAME/Orbitchat-Gemini-AI.git`
2. **Install requirements:** `pip install -r requirements.txt`
3. **Set your API Key:** Add your `GOOGLE_API_KEY` to your secrets/environment.
4. **Run the app:** `streamlit run app.py`
